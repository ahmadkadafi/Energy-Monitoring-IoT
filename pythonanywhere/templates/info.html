Deskripsi Sistem IoT

Sistem yang dirancang ini, merupakan sistem monitoring energi listrik berbasis Internet of Things (IoT) yang berfungsi untuk memantau parameter kelistrikan secara real-time dan historis. Sistem ini dikembangkan sebagai bagian dari Tugas Besar mata kuliah IoT Sistem Cerdas pada program Magister Teknik Elektro, Telkom University. Data kelistrikan diukur menggunakan sensor PZEM-004T, kemudian diproses oleh mikrokontroler dan dikirimkan ke server melalui jaringan internet. Data yang dikirim meliputi tegangan, arus, daya, energi, frekuensi, dan faktor daya. Seluruh data disimpan dalam basis data dan divisualisasikan melalui web dashboard berbasis Flask.

Perangkat & Teknologi
Komponen Spesifikasi / Fungsi

Mikrokontroler ESP32 / NodeMCU (IoT communication)
Protokol Komunikasi UART (sensor) & HTTP REST API
Backend Python Flask
Database MySQL (PythonAnywhere)
Frontend HTML5, Bootstrap 5, JavaScript, Chart.js
Fitur Sistem Monitoring real-time, grafik historis, log data, filter, export data, dan visualisasi lokasi perangkat
